Welcome to H3LG3.

This a robot project for Raspberry newbies:

1. Get Python and Ruby up on RPI
2. Camera running
3. Arduino robot movement controller
4. Node webserver
5. Free domain linking
6. Remote controls
7. Weapons
8. Picture analysis
9. Trigger remote alarms
10. Android app
11. Robots homepage

Vision

A surveillance robot that will host it self and allow full control wherever it will be stationed.

Needs

A battery that is light weight and powerful. Night vision or headlamps. Chassis
